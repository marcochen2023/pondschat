# 🤝 貢獻指南

感謝你對 PondsChat 的興趣！我們歡迎任何形式的貢獻。

---

## 📋 貢獻流程

### 1. Fork 專案

點擊 GitHub 頁面右上角的 "Fork" 按鈕。

### 2. 克隆你的 Fork

```bash
git clone https://github.com/YOUR_USERNAME/pondschat.git
cd pondschat
```

### 3. 建立分支

```bash
git checkout -b feature/your-feature-name
# 或
git checkout -b fix/bug-fix-description
```

### 4. 開發與測試

```bash
# 啟動開發伺服器
npm start

# 訪問 http://localhost:3003
```

### 5. 提交變更

```bash
git add .
git commit -m "Add: 描述你的變更"
```

### 6. 推送分支

```bash
git push origin feature/your-feature-name
```

### 7. 建立 Pull Request

前往原始專案頁面，點擊 "New Pull Request"。

---

## 📝 程式碼規範

### JavaScript
- 使用 ESLint 標準
- 避免全域變數污染
- 使用有意義的變數名稱
- 註解複雜邏輯

### CSS
- 優先使用 Tailwind CSS 類別
- 自定義樣式放在 `public/css/style.css`
- 保持響應式設計

### HTML
- 語義化標籤
- 可訪問性（Accessibility）

## 🐛 報告問題

### 報告 Bug
1. 前往 [Issues](https://github.com/yourusername/pondschat/issues)
2. 點擊 "New Issue"
3. 選擇 "Bug Report"
4. 填寫問題模板

### 功能建議
1. 前往 [Issues](https://github.com/yourusername/pondschat/issues)
2. 點擊 "New Issue"
3. 選擇 "Feature Request"
4. 描述你的想法

---
